# Exetnsion chrome timer de concentration pour pronote 

Comment installer ?

1. Télécharger la dernière realease
2. Dans chrome aller dans extension
3. Activer le mode délevoppeur
4. Cliquer sur charger l'extention non empaqueter
5. Selectionner le dossier de l'extension
6. Cliquer sur Ok

Et voilat !
